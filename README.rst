TakingItCasual.github.io
========================

My reverse-engineering of the puzzle game `TIS-100`_ by Zachtronics_.
The font tis100copy was created with screenshots of the game and BitFontMaker2_.
Still a work in progress (it's basically just at text editor at the moment).
Visit `TakingItCasual.github.io`_ to see it in action.

.. _TIS-100: http://store.steampowered.com/app/370360/TIS100/

.. _Zachtronics: http://www.zachtronics.com/

.. _BitFontMake2: http://www.pentacom.jp/pentacom/bitfontmaker2/

.. _TakingItCasual.github.io: https://TakingItCasual.github.io
